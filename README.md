## Стэк в условии задачи: Vue (Composition API) + Pinia + Typescript. Так же, в условии список userList.json

## Project Setup

```sh
yarn
# or
npm install
```

### Compile and Hot-Reload for Development

```sh
yarn dev
# or
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
yarn build
# or
npm run build
```
